Integrating With Note Taking Generally

 

 

An easy way to mirror or fork this wiki is by cloning this GitHub repository and then using these shell scripts to search/find/tag/link/preview all the files and directories, this works very well with a combination of guake and Vim/Emacs/VSCode/Marktext/Zettlr .

 

Another option is cloning the repository and using:

  • VNote
    • Litters the Directory with json but otherwise really good.
  • Joplin
    • uses a SQLite DB
  • Notable
    • Electron App, closed source though.
      • but, dev contributes free libraries and the structure of the app is zero lock in
